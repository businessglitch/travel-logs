# Travel-logs
A simple MERN stack app where you can log all places you have visited. Uses MapBox to display the map. 

# Instruction
Install yarn
run `yarn install` in server folder.
# Log Entry Schema
 * Title
 * Date Visited
 * Comment
 * Start Date
 * End Date
 * Latitude
 * Longitude
 * Description
 * Image
 * Created At
 * Updated At
